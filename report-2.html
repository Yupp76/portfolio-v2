Rainydays

Rainydays is a fictional ecommerce website that I enjoyed designing & building it. Changes made to this projet come as the following:

Add to cart functionality

I used JavaScript to add a new functionality for the add to cart button in the product details page. this functionality will make it possible for the user to chose an amount of items and add them all to his cart. This will be positive for the user experience.

Improve accesibility

I updated the colors of the text and buttons to improve their contrast with their backgrounds. this will help users with accesibility issues to use the website with ease.

Community Science Museum

Community science museum was a beautifull project that I was happy to work on. This project is a concept website for a fictional mueseum that holds diffrent kinds of activities like exhibitions and events. Changes made to this projet come as the following:

Improve SEO & accesibility

I removed empty meta tags, and added alt to img tags. This improves the website's SEO and makes it more accesible for users as well as improving the user experience.

Add JavaScript

I added JavaScript to this website to handle the hamburger menu functionality and make the whole page unscrollable after the sidebar is open. This makes it easy for the user to complete his navigation around the website pages without distraction.
